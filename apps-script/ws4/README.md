# WS4 — Calculator Database / logger (`WS4_LOG` in `app.js`)

Handles `logQuote`, `sendQuote`, and related actions.

1. Copy `.clasp.json.example` → `.clasp.json` and set **Script ID** from this project’s Apps Script **Project settings**.
2. From repo root: `npm run clasp:pull:ws4` then edit `.gs` files here.
3. Push: `npm run clasp:push:ws4`
